Live Sports Companion

For the React Native challenge during HackGT 12, I fully designed and developed this project

Overview

This app transforms the sports experience to be more interactive, allowing users to engage more with their parlays. The core features include live sports coverage and updates, game predictions, interactive polls, watch parties to enjoy sports with friends, and key highlights which users can watch.

Design

I used Figma to visualize and create high fidelity screens showing the placement of various desired features and an efficient flow.

Since this app serves as a companion to PrizePicks to allow users to engage with the parlays they created in the existing app, I decided to match the design style so that users would immediately feel a sense of familiarity and continuity between the two experiences, making it easier to navigate, trust, and transition into this live sports app without needing to relearn the interface or visual language.
